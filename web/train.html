4Export Model & Upload to S3

Downloads the trained model files and the labels JSON. Upload both to your S3 bucket, then paste the model.json URL into the Jestally extension settings.

After downloading:
1. Run: aws s3 cp model/ s3://jestally-isl-gifs-prod/model/ --recursive --region ap-south-1
2. Run: aws s3 cp labels.json s3://jestally-isl-gifs-prod/model/labels.json --region ap-south-1
3. Paste this URL into the extension Model URL field:
