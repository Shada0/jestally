System Architecture  ·  AWS Native  ·  Serverless

Built to Scale

👁️

On-Device Hand Tracking

MediaPipe extracts a 21-point hand skeleton entirely in-browser. Zero video leaves the device — only a compact 63-float coordinate vector is transmitted for inference.

⚡

Phrase-First Resolution

The translation engine resolves multi-word expressions before word-level lookup — delivering contextually accurate, natural ISL sequences every time.

🌐

All 22 Scheduled Languages

Speech input and sign-to-speech output across all 22 constitutionally recognised Indian languages — Hindi, Tamil, Telugu, Bengali, Kannada, Malayalam, and beyond.

🔒

Security by Design

Least-privilege IAM at every service boundary. API key authentication on all endpoints. CORS scoped to authorised origins. No user data stored.

🧩

Live Video Call Integration

A Manifest V3 Chrome extension embeds Jestally directly into Google Meet and Zoom — an always-on ISL overlay with zero performance impact on the call.

🎮

Production ML Pipeline

TensorFlow Serving on Amazon SageMaker delivers low-latency inference at scale. Custom pre/post-processing normalises hand geometry and returns confidence-ranked predictions.

Infrastructure as Code  ·  One-Command Deploy

Deploy in Minutes

A single CloudFormation template provisions the entire stack — API Gateway, Lambda, IAM, S3, and SageMaker. Repeatable across environments with zero manual steps.

01

Provision Infrastructure

One template deploys API Gateway, Lambda, IAM roles, and S3 in a single atomic operation — fully repeatable across environments.

aws cloudformation deploy …
02

Deploy the ML Model

Package the trained model, upload to S3, and provision a SageMaker TF Serving endpoint — fully scripted, no manual steps.

python create_sagemaker_model.py
03

Deploy Lambda Handler

Zip, deploy, and wire the inference Lambda to API Gateway with a single command. Permission bindings are handled automatically.

python deploy_lambda.py
04

Connect the Frontend

Paste your endpoint and API key into the Configure API panel. The full system goes live immediately — no rebuild required.

Product Roadmap

Speech-to-ISL Translation & Sign Recognition

Web application and Chrome extension delivering real-time ISL translation, on-device hand tracking, AWS-hosted sign classification, and multilingual TTS — production-ready and live.

Continuous Streaming & Sentence-Level Understanding

WebSocket-based sign streaming for uninterrupted full-sentence recognition. Fingerspelling fallback for vocabulary edge cases.
